# ☁️ My Cloud

## Описание

**My Cloud** — это персональное облачное хранилище, разработанное на **FastAPI** с использованием **Google Drive API** в качестве файлового хранилища.

Проект позволяет загружать, просматривать, скачивать и удалять файлы через удобный веб-интерфейс, при этом сами файлы физически хранятся на Google Drive.

На данный момент проект предназначен для личного использования и работает через OAuth-авторизацию одного Google-аккаунта.


---

#  Возможности

-  Загрузка файлов напрямую в Google Drive
-  Просмотр файлов из выделенной папки Google Drive
-  Скачивание файлов
-  Удаление файлов
-  Хранение файлов в отдельной папке Google Drive
-  Авторизация через Google OAuth 2.0
-  Современный веб-интерфейс

#  Установка

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/intelnohd/localcloud.git
cd localcloud
```

### 2. Установите зависимости

```bash
pip install -r requirements.txt
```

### 3. Создайте проект в Google Cloud

Необходимо:

- включить Google Drive API;
- создать OAuth Client;
- скачать `credentials.json`;
- поместить его в папку проекта.

При первом запуске откроется окно авторизации Google и автоматически создастся файл `token.json`.

---

### 4. Запуск

```bash
uvicorn main:app --reload
```

После запуска откройте

```
http://127.0.0.1:8000/files
```
